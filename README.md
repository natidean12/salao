# Salão
# 📌 Sobre o projeto

O Salão & Barbearia Premium é uma landing page institucional desenvolvida para apresentar os serviços de um salão de beleza e barbearia moderna, oferecendo uma experiência elegante, sofisticada e totalmente responsiva. O projeto foi criado com foco em destacar a qualidade dos serviços, facilitar o agendamento online e transmitir credibilidade por meio de um design premium e interativo.

A aplicação reúne diversas funcionalidades que tornam a navegação dinâmica, como navegação horizontal em tela cheia, galeria interativa de antes e depois, tabela completa de serviços e preços, apresentação da equipe, formulário de agendamento, integração com WhatsApp, perguntas frequentes (FAQ) e diversos efeitos visuais modernos.

Toda a interface foi desenvolvida utilizando um layout contemporâneo com Glassmorphism, animações suaves e responsividade para computadores, tablets e smartphones, proporcionando uma excelente experiência em qualquer dispositivo.

# 🎯 Objetivo

Desenvolver uma landing page moderna e sofisticada para salões de beleza e barbearias, apresentando seus serviços de forma profissional, facilitando o agendamento online e proporcionando uma experiência visual elegante, interativa e totalmente responsiva.
